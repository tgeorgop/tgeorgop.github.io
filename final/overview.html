What is H5P?

H5P allows instructors to create multimedia-rich lessons, interactive videos that include quiz questions and navigation links, knowledge checking, and other content types – all of which can be embedded in a Canvas course. Once created within your Canvas course, these activities can be cloned for reuse in your other Canvas courses, without any need to recreate them. H5P content is responsive and mobile friendly!
